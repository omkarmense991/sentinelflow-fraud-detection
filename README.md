# SentinelFlow

Production-style Machine Learning Engineering project for financial fraud detection.

## Overview

SentinelFlow is a scalable fraud detection platform designed to simulate real-world ML engineering workflows used in financial systems.

The project evolves through multiple stages:

1. Traditional ML fraud detection pipelines
2. Production-grade inference services
3. Graph Neural Network based relational fraud detection

## Objectives

- Learn production ML engineering workflows
- Understand fraud detection systems
- Handle severe class imbalance
- Build modular training pipelines
- Deploy realtime fraud prediction APIs
- Explore graph-based fraud analysis

## Tech Stack

- Python
- Scikit-learn
- XGBoost
- FastAPI
- PostgreSQL
- Docker
- PyTorch Geometric
